== Installation ==

1. `mac-sonuclari` klasörünü `/wp-content/plugins/` dizinine yükleyiniz
1. Maç Sonuçları eklentisiniz wordpress admin menüsündeki 'Eklentiler' menüsünde aktive ediniz.
1. Ayarlar menüsünü yükleme sonrası ziyaret edebilirsiniz.
1. [macsonuclari] - Bu kodu wordpress sayfa yada postunun içine istediğiniz yere kopyala/yapıştır yaparak canlı skorları sayfanızda gösterebilirsiniz.
